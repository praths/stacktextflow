The stackoverflow editor you'll love.

A browser plugin that replaces the default stackoverflow editor with a powerful WYSIWYG editor.

Learn more

Features

See the format you're applying

stacktextflow is a WYSIWYG editor so of course you see the formatting you choose right there in the editor. Make text bold or italic, add headings and blockquotes, insert code and pictures and hyperlinks.

Don't search, add a link

Select a command, a class or anything you would like to link to click on the link button and stacktextflow will try to provide you a suggestion. NSString, Java String, Count Python, DuckDuckGo knows for nearly everything a quick link.

Code on the fly

We built in the Ace code editor. Writing code on stackoverflow has never been so easy before. Syntax highlighting, suggestions and syntax error warnings makes coding as easy as in your favorite editor.

...and much more.

Download

You can download the alpha plugin for Safari. This is just a preview and we really don't recommend using it for answering questions.

We need your help!

We need your help to create stacktextflow! Everybody who likes the idea of stacktextflow is invited to contribute. Only if a lot of people help this project can be realized.

stacktextflow is completely written in JavaScript, if you want to contribute to the code you have to know JavaScript. If you don't know JavaScript you can help anyway because we also need people creating styles, testing the plugin and reporting bugs.

Read CONTRIBUTE.MD in our GitHub repository to learn more.
